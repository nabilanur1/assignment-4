## Answers to Questions

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
Ans:The main differences lie in their selection mechanism , the number of elements they return, and whether the returned collection is live or static.

### 2. How do you create and insert a new element into the DOM?
Ans:To create and insert a new element into the DOM, i first create the element using document.createElement(), then configure it, and finally append it to an existing element in the document using methods like appendChild(), append(), prepend(), or insertBefore().
